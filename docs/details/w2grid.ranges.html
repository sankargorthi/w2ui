Array of all ranges defined for the grid.
Array, default = []
This property only works if you grid is excel-like grid, .selectType is set to 'cell'. To add and remove ranges you should use .addRange and .removeRange methods. If can find a range by name using .getRange method. If you need to refresh ranges, use .refreshRanges However, if you want to see what ranges are defined, you may iterate through .ranges property
